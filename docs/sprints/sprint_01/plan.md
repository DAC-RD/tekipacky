# Sprint 01 計画

## 概要

| 項目 | 内容 |
|------|------|
| 期間 | 2026/02/26 |
| テーマ | 要件定義・UIモック・Next.js実装 |
| ゴール | ローカルストレージで動作するNext.jsアプリとしてダッシュボードを実装する |

---

## スプリントの目標

- アプリの要件を定義する
- ダッシュボードと追加・編集モーダルのUIをモックで検証する
- モックをベースにNext.jsで実装し、ローカルストレージでデータを永続化する
