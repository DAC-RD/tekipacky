# 要件定義

## プロダクト概要

**名称:** テキパッキー (tekipacky)  
**目的:** 脳疲弊によって日常行動に着手できない人が、小さな行動を積み重ねられるようにする

---

## ユーザーストーリー

- 風呂に入る・食器を洗うといった日常行動を「行動メニュー」として登録し、実行するたびにポイントを獲得できる
- 貯めたポイントを消費して、動画視聴・外食などの「ご褒美行動」と交換できる
- ポイントの収支を見ることで、今日どれだけ動けたかを把握できる

---

## 機能要件

### 行動メニュー

登録できる項目は以下の通り。

| 項目 | 選択肢 |
|------|--------|
| タイトル | テキスト入力 |
| 説明 | テキスト入力（任意） |
| タグ | 任意の文字列、複数設定可 |
| ハードル（面倒くささ） | 低(×1) / 中(×2) / 高(×3) |
| かかる時間 | 5分以内(×1) / 15分以内(×2) / 30分以内(×3) / 1時間以内(×4) / 3時間以内(×5) / 3時間以上(×6) |

獲得ポイント = ハードル × 時間 × モード倍率

### ご褒美メニュー

登録できる項目は以下の通り。

| 項目 | 選択肢 |
|------|--------|
| タイトル | テキスト入力 |
| 説明 | テキスト入力（任意） |
| タグ | 任意の文字列、複数設定可 |
| 満足度 | 小(×1) / 中(×2) / 大(×3) |
| かかる時間 | 行動メニューと同じ6択 |
| 金額 | 0円(×1) / 500円以内(×2) / 1000円以内(×3) / 5000円以内(×4) / 1万円以内(×5) / 1万円以上(×6) |

消費ポイント = 満足度 × 時間 × 金額 × モード倍率

### ポイント管理

- ユーザーはアカウント全体でモードを設定できる
- モードによってポイントの獲得・消費の倍率が変わる

| モード | 獲得倍率 | 消費倍率 |
|--------|----------|----------|
| イージー | ×1.5 | ×0.7 |
| ノーマル | ×1.0 | ×1.0 |
| ハード | ×0.8 | ×1.5 |

- ポイントが不足している状態でご褒美を使用することはできない
- 将来的に過去のポイント推移・行動統計を参照できる画面を設ける

### ダッシュボード

- 上部に現在のポイント・今日の獲得ポイント・今日の消費ポイントを表示する
- 下部に行動リスト・ご褒美タブを切り替えて表示する
- 各タブにはタグによるフィルター機能を設ける
- 行動カードをタップするとポイントが加算され、浮き上がりエフェクトが出る
- ご褒美カードをタップするとポイントが消費される
- 行動・ご褒美メニューはタップしても一覧から消えない（繰り返し実行を前提とした設計）
- アコーディオン形式で「今日の行動ログ」「今日のご褒美ログ」を折りたたみ表示する
- 今日のログには `食器を片付ける × 3` のように実行回数をまとめて表示する
- ログの回数は編集可能（誤操作の取り消しに対応）
- 日付が変わると今日のログはリセットされる（ポイント残高は持ち越し）

### 追加・編集

- 右上（タブバー横）の＋ボタンから追加モーダルを開く
- モーダル内で行動・ご褒美を切り替えて登録できる
- 設定した内容に基づいてポイントのプレビューをリアルタイム表示する
- 各カードの編集ボタン（✎）から編集・削除ができる
- タグ欄では既存タグをサジェスト表示し、クリックで追加・解除ができる

---

## 非機能要件

- スマートフォン・PCの両方で使えるレスポンシブデザイン
- 認証機能は後回し（フェーズ1はローカルストレージで動作）

---

## 対象外（将来対応）

- ユーザー認証
- データベース永続化
- 過去統計・グラフ表示
- 通知機能
