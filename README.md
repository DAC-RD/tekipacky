# テキパッキー (tekipacky)

テキパキと行動できるようになるための行動管理アプリ。

脳疲労によってやる気が枯渇している人が、風呂に入る・食器を洗うといった日常の行動にすら着手できない問題を解決する。行動をポイントに換算して少しずつ積み上げ、貯めたポイントでご褒美行動と交換できる仕組みにより、行動の動機づけをゲーム的に設計する。

## リポジトリ構成

```
tekipacky/
├── apps/
│   └── web/                  # Next.js アプリ
├── docs/
│   ├── requirements/         # 要件定義
│   └── sprints/              # スプリントごとの計画・設計・レビュー
│       └── sprint_01/
│           ├── plan.md
│           └── mock/
└── README.md
```

## 開発方針

息切れしないよう小さく進める。まずローカルで動くUIを作り、DBや認証は後から追加していく。

## 技術スタック（予定）

- フロントエンド: Next.js
- データベース: PostgreSQL
- ホスティング: AWS
